## 运行 TtsMain，进行合成



结果在result.mp3，如果遇见错误，显示在System.err内

其中

- Content-Type: audio/mp3，表示合成成功，可以播放MP3 result.mp3
- Content-Type: application/json 表示错误 。可以看到错误信息的json

   // text 的内容为"欢迎使用百度语音合成"的urlencode,utf-8 编码
